# Open File In New Terminal Tab

Swift port of Objective-C app - a status bar wrapper around AppleScript to open your current finder directory in a new terminal tab.

## Dependencies

* Swift 1.2
* OSX 10.10
* XCode 6.3.2

## Getting Started

* Open in XCode
* Product -> Archive -> Export
